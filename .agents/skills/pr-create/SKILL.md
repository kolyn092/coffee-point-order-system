---
name: pr-create
description: 현재 브랜치 변경을 실제 커밋과 변경사항을 기준으로 한국어 PR 제목·본문을 작성해 GitHub에 등록한다. 
  사용자가 "PR 만들어줘", "PR 올려줘", "풀리퀘스트 작성해줘"처럼 현재 변경의 PR 생성을 요청할 때 사용한다. 
  코드 내용은 수정하지 않는다.
---

## 기본 규칙

- PR 제목은 `[type] 한국어 요약` 형식을 사용한다.
- PR 본문 구조의 정본은 `.github/PULL_REQUEST_TEMPLATE.md`이다. 해당 템플릿을 찾지 못한 경우 아래 템플릿을 사용한다.
- 확인된 관련 이슈가 있으면 PR 본문에 이슈마다 `Closes #번호` 형식으로 링크한다. 
- 리뷰 반영으로 추가된 수정 사항과 관련된 이슈도 같은 형식으로 추가하며, 이슈 번호를 추측하지 않는다.
